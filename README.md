# GREEN-API Test

Простая одностраничная HTML-страница для вызова методов GREEN-API:

- getSettings
- getStateInstance
- sendMessage
- sendFileByUrl

## Запуск локально

Достаточно открыть `index.html` в браузере.

Если браузер ограничивает CORS для локального файла, поднимите простой статический сервер, например:

```bash
python3 -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Публикация исходников

1. Создать новый репозиторий на GitHub.
2. Загрузить в него файлы из этой папки.
3. При необходимости включить GitHub Pages или задеплоить на любой статический хостинг.
